# UI And UX Direction

The app should feel calm, warm, and motivating. It should feel closer to a personal reading space than a competitive productivity tracker.

## Design Goals

- Simple dashboard
- Fast check-ins
- Clear reading progress
- Encouraging language
- Low-pressure reminders
- Mobile-first layout
- Clean typography
- Minimal clutter
- Visible group activity
- Strong spoiler protection

## Notification Strategy

Notifications should help users return to reading without overwhelming them.

Recommended notifications:

- Monday check-in reminder
- Thursday discussion reminder
- Friend replied to your post
- Friend finished a book
- Book of the Month voting opened
- Weekly recap ready

Avoid:

- Constant streak pressure
- Multiple reminders per day
- Guilt-based wording
- Aggressive push notifications
