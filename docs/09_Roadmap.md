# Roadmap

## Phase 1: Foundation

Build authentication, profiles, groups, current books, and reading logs.

## Phase 2: Habit Loop

Build check-ins, streaks, basic XP, and weekly reminders.

## Phase 3: Social Reading

Build discussion posts, replies, group feed, and reactions.

## Phase 4: AI Prompts

Build AI-generated questions, prompt history, and spoiler-aware prompt design.

## Phase 5: Reviews

Build live reviews, final reviews, category ratings, and review timelines.

## Phase 6: Fair Scoring

Build Reading Momentum, XP events, achievement tracking, and leaderboards.

## Phase 7: Book Of The Month

Build nominations, voting, finalists, and optional group reading.

## Phase 8: Polish

Improve UI, onboarding, accessibility, analytics, notifications, and PWA support.

## Phase 9: Native App Evaluation

Only consider a native mobile app after proving weekly retention.
