# Reading Momentum

Reading Momentum is a social reading accountability web app. It helps friends keep a consistent reading habit while allowing each person to read their own book, format, goal, and pace.

## Current Sprint

Sprint 1: Project Foundation And Infrastructure

Scope:

- App shell
- Navigation
- Auth page scaffolding
- Profile setup scaffolding
- Supabase configuration placeholders
- Testing and deployment preparation

Out of scope:

- Reading Momentum scoring
- XP
- Achievements
- Leaderboards
- AI prompts
- Reviews
- Notifications
- Book of the Month

## Setup

Install dependencies:

```bash
pnpm install
```

Run locally:

```bash
pnpm dev
```

Environment variables are listed in `.env.example`.
